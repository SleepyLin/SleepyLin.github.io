Github博客搭建

环境准备

git安装：https://git-scm.com/
node.js安装：https://nodejs.org/en/download/
创建github仓库：username.github.io
git中安装hexo： npm install hexo -g

SSH Key设置

打开git： Git bash
进入ssh目录： cd ~/.ssh
生成rsa公钥： ssh-keygen -t rsa -C "@youremail"
复制rsa公钥： Windows: C:/user/Administrator/.ssh/id_rsa.pub
GitHub配置：Setting/SSH and GPG keys/New SSH Key
git配置

Hexo搭建blog

创建Blog目录 ./MyBlog
git中初始化个人博客：hexo init
在./MyBlog 中编辑 _config.yml

编写blog

git中新建博客：hexo new 'blog-title'
编辑./MyBlog/source/_posts/'blog-title.md'文件创作blog
git中部署blog至github

查看blog

https://username.github.io/
